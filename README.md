# vue-study-note

## 프로젝트 소개
생성,수정,삭제,조회가 가능한 학습노트 제작 프로젝트 입니다.

> 해당 글의 내용은 [인프런] Vue.js 끝장내기 - 실무에 필요한 모든 것 강의를 토대로 만들어졌습니다.

<br />

## 프로젝트 설치 및 실행

### Front 서버

#### 설치
```
npm install
```

#### 개발모드 실행
```
npm run serve
```

#### 운영모드
```
npm run build
```

#### 테스트 실행 
```
npm run test
```

<br />

### Backend 서버

#### 설치
```
# server 파일로 이동
cd server

# node 버젼 10.13.0
nvm install 10.13.0

# 설치
npm install
```

#### 실행
```
npm run dev
```
